Métricas Importantes (Platformer Kit)

                                
    
                                    
  • Grade X (Largura): Cada bloco # ou rampa / mede 2m.
    • 
                                    
  • Grade Y (Altura): Cada linha mede 3m (gravado como ystep: 3.0 no arquivo).
    • 
                                    
  • Salto Médio: 4m horizontalmente (parado), até 15m (com velocidade máxima).
    • 
                                    
  • Molas: Verticais lançam a 22m de altura. Diagonais para cruzamento rápido.
    • 
                                    
  • Queda Fatal: Evite plataformas em Y < -15m.
    •
