Important Metrics (Platformer Kit)

                                
    
                                    
  • X Grid (Width): Each block # or ramp / measures 2m.
    • 
                                    
  • Y Grid (Height): Each row measures 3m (saved as ystep: 3.0 in the file).
    • 
                                    
  • Average Jump: 4m horizontally (standing), up to 15m (at maximum speed).
    • 
                                    
  • Springs: Vertical ones launch up to 22m high. Diagonal ones for fast crossings.
    • 
                                    
  • Fatal Fall: Avoid platforms at Y < -15m.
